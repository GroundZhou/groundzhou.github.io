## 2.4 UFW

[官方帮助手册](https://help.ubuntu.com/community/UFW)

__注：需要root权限。__

```sh
ufw enable/disable # 开启/关闭防火墙
ufw allow 22/tcp # 添加规则：允许22/tcp端口
ufw deny 22/tcp # 添加规则：拒绝22/tcp端口
ufw status numbered # 查看防火墙状态以及带有序号的规则列表
ufw delete RULE|NUM # 删除规则
ufw reset # 重置防火墙
ufw reload # 重载防火墙
```
